Launch Run.java

Edit map: 
- Place and delete blocks
- A door must be place after every key
- Eraser deletes blocks

Glue Maps: Select 2 maps to be displayed side by side. Once selected, you can place objects in either map,
 including keys that link to doors in different rooms.

Load map: Loads a .xml file, sets it back to default mode with only 1 map being displayed

Clear map: Clears the screen

Save map: Exports the map to a .xml file where you want, override files or save your own. When in glued mode it saves the 
first map, then the second.
